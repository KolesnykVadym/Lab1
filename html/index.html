Що таке HTML?

HTML (від англ. HyperText Markup Language — 'мова гіпертекстової розмітки') — стандартизована мова розмітки документів у Всесвітній павутині.
HTML підручник

HTML Елементи

HTML елемент зазвичай складається із тегу початкового і тегу кінцевого, а зміст вставляється поміж ними. Порожні елементи не мають кінцевого тегу.
HTML підручник

HTML Атрибути

Всі HTML елементи можуть мати атрибути. Вони надають додаткову інформацію про елемент завжди вказуються в початковому тезі. Атрибути зазвичай входять в пари ім'я/значення.

HTML Посилання

HTML посилання - це гіперпосилання. Ви можете натиснути на посилання і перейти до іншого документу. При наведенні миші на посилання стрілка миші перетвориться в маленьку стрілку.

HTML Зображення

Зображення можуть поліпшити дизайн і зовнішній вигляд веб-сторінки. В HTML зображення визначаються за допомогою тега *img*
HTML підручник

HTML Класи

HTML атрибут class використовується для визначення класу для HTML елемента. Кілька елементів HTML можуть спільно використовувати один і той же клас.

CSS підручник

Що таке CSS?

CSS (англ. Cascading Style Sheets, рус. Каскадні таблиці стилів) — спеціальна мова, яка використовується для опису зовнішнього вигляду сторінок, написаних мовами розмітки даних.
CSS підручник

CSS Синтаксис

Набір правил CSS складається із селектора та блоку об’яви. Селектор вказує на елемент HTML, який ви хочете стилізувати. Блок об’яв містить одну або кілька об’яв, розділених крапкою з комою.
CSS підручник

CSS Селектори

CSS-селектор вибирає HTML-елементи, до яких потрібно додати стиль. використовуються для "пошуку" (або вибору) HTML-елементів, які ви хочете стилізувати.

CSS підручник

CSS Таблиці

Зовнішній вигляд HTML таблиці можна значно покращити за допомогою CSS. Щоб вказати межі таблиці в CSS, використовуйте властивість border.
CSS підручник

CSS Блочна модель

Усі елементи HTML можна розглядати як блоки (коробки). Всі елементи HTML можна розглядати як блоки. В CSS термін "блочна модель" використовується, коли говорять про дизайн і верстку.
CSS підручник

CSS Шрифти

Вибір правильного шрифту для вашого сайту - це дуже важливо! Дуже важливо використовувати шрифт, який легко читається. Шрифт додає цінність вашому тексту. Також важливо обрати правильний колір і розмір тексту для шрифту.
